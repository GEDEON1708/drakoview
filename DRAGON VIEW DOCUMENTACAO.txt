# Documentação do Sistema de Visualização de Dragões

## Resumo do Sistema

O sistema de Visualização de Dragões foi desenvolvido para permitir que usuários busquem informações sobre diferentes dragões com base no seu nome. Ele apresenta uma interface gráfica amigável, onde o usuário pode inserir o nome do dragão desejado e visualizar detalhes como descrição, idade, peso, montador e a casa a qual pertence. O sistema está preparado para futuras atualizações e ajustes conforme necessário, garantindo uma experiência ainda melhor para os usuários.

## Instruções para Digitação de Dados

Ao utilizar o sistema, o usuário deverá inserir dados conforme os seguintes critérios:

1. **Campo de Busca**:
   - **Nome do Dragão**: O nome deve ser digitado exatamente como está cadastrado. O sistema não aceita variações. Por exemplo, se um dragão se chama "Drogon", digitar "Drogon" (com letras maiúsculas e minúsculas corretas) é o único jeito de encontrá-lo.
